# Contributing

Thanks for helping improve ComfyUI-RUM.

## Development checklist

1. Install the node into a ComfyUI `custom_nodes` directory.
2. Install dependencies in the same Python environment ComfyUI uses.
3. Run `python scripts/check_install.py`.
4. Test a small workflow before opening a PR.

## Bug reports

Please include:

- ComfyUI version
- Python, PyTorch, diffusers, transformers versions
- GPU and VRAM
- Full traceback
- Node settings and prompt
